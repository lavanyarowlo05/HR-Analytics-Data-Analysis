# HR Analytics Dashboard – Employee Attendance Insights

## Project Overview

This project focuses on analyzing employee attendance patterns using data analytics and visualization techniques. The goal of the project is to help HR teams understand workforce availability, identify attendance trends, and support data-driven decision making for improving productivity and workforce planning.

Using historical attendance data, this project builds an interactive dashboard that highlights key HR metrics such as employee presence, work-from-home trends, and leave patterns.

Business Problem

Organizations often struggle to monitor employee attendance patterns effectively. Without proper analysis, HR teams cannot easily identify issues such as high absenteeism, excessive leaves, or productivity loss.

The objective of this project is to analyze attendance data and generate actionable insights that help HR departments:

monitor workforce availability

track employee attendance behavior

identify trends in work-from-home and leave usage

improve workforce planning and productivity

## Dataset

The dataset contains employee attendance information including:

Employee ID

Attendance status

Work from home records

Leave records

Presence percentage

The data was processed and transformed to generate meaningful HR insights.
